03 — Технологии

Передовые технологии

Инновации, которые делают RTX 6070 Ti лидером.

🤖

DLSS 4 Multi Frame

Генерация нескольких кадров из одного с помощью ИИ. До 8x прирост FPS без потери качества изображения.

💫

Ray Tracing 3-го поколения

Реалистичное отражение света, тени и глобальное освещение. Играйте с полной трассировкой лучей в 4K.

⚡

NVIDIA Reflex 2

Минимальная задержка ввода. Синхронизация GPU и CPU для максимально отзывчивого геймплея.

AV1 кодирование

Аппаратное кодирование AV1 для стримеров. Лучшее качество при меньшем битрейте.

🧬

AI-ускорение

276 Tensor ядер 4-го поколения для генерации изображений, нейросетей и машинного обучения.

DisplayPort 2.1

Мониторы 4K @ 240Hz или 8K @ 60Hz без сжатия сигнала.
